# Descrição
Portal de gerenciamento de projetos, publicações e eventos do grupo de pesquisas TAG.

---
# Instalação do ambiente de desenvolvimento
##### Requisitos
- Python 3.x
- pip3
- Node 13.x
- npm compatível com Node 13.x
#### Instalação do Servidor
- Na branch dev-backend
- Na pasta TAGserver
- execute: ```pip install -r requirements.txt```
- inicialize o servidor com: ```python manage.py runserver```
#### Instalação do Client
- na branch dev-frontend
- na pasta TAGclient/portal-tag
- execute: ```npm i```
- inicialize o client com: ```npm start```
